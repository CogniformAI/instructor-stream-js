# Release 0.5.1

## Summary

- Streaming snapshots now emit `{ data, _meta }` across the core runtime and adapters.
- Documentation and examples align with the `_meta` output shape.

## Changes

- Emit `_meta` in Effect streaming snapshots and propagate through langgraph/openai adapters.
- Update tests to assert `_meta` instead of `meta`.
- Refresh readme and adapter docs to match the new streaming format.
